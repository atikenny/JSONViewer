# JSONViewer

JSONViewer Snippet you can run in Chrome

## Features

### Basics
- [x] skip empty items
- [ ] folding arrays and objects
- [ ] show parsing errors

### Styling
- [x] inject your own stylesheet
- [ ] dark mode
- [ ] monokai themes
- [ ] theme selector

### Advanced
- [ ] search input
- [ ] filter values for a given key

### Accessibility
- [ ] keyboard shortcuts
- [ ] help modal with keyboard shortcuts
